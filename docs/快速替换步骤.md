# 快速替换步骤

## 如果已安装 skill

```bash
# 进入 skill 目录
cd ~/.openclaw/skills/cc-supervisor

# 拉取最新代码
git pull origin main

# 验证版本
cat SKILL.md | grep "version:"
# 应该显示: version: 0.7.2
```

## 如果是本地开发

```bash
# 在项目根目录
cd /Users/xz/Projects/cc-supervisor

# 提交更改
git add .
git commit -m "feat: add comprehensive autonomous decision rules (v0.7.2)

- Add AUTONOMOUS_DECISION_RULES.md with full decision trees
- Add 自主决策规则总结.md for quick reference
- Strengthen SKILL.md trigger rules (MANDATORY)
- Update autonomous mode to fully autonomous (auto-approve all programming ops)
- Increase round limits (20→30, 5→8, 2→3)
- Add UPGRADE_GUIDE.md"

# 如果需要推送
git push origin main

# 如果已安装到 ~/.openclaw/skills/，更新
cd ~/.openclaw/skills/cc-supervisor
git pull origin main
```

## 验证

```bash
# 检查新文件
ls -la docs/AUTONOMOUS_DECISION_RULES.md
ls -la docs/自主决策规则总结.md
ls -la docs/UPGRADE_GUIDE.md

# 检查触发规则
grep "MANDATORY" ~/.openclaw/skills/cc-supervisor/SKILL.md
```

## 测试

```bash
# 测试触发
openclaw agent --message "请使用 Claude Code 实现一个 web 服务器"
# agent 应该立即调用 cc-supervisor skill

# 测试自主模式
OPENCLAW_SESSION_ID=<session-id> CC_MODE=autonomous cc-supervise /path/to/project
cc-send "创建项目，安装依赖，运行测试，提交代码"
# agent 应该自动批准所有操作，不停下来问人
```

## 关键改进

1. **触发规则强制化** - 使用 MANDATORY/MUST/IMMEDIATELY，防止 agent 忘记
2. **完全自主模式** - 所有编程操作自动批准，默认答案是 `y`
3. **极少升级** - 只在真正卡住时升级（缺信息、重复失败 3 次、系统错误）

详细说明见 `docs/UPGRADE_GUIDE.md`
